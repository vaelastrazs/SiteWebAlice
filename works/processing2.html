Code Processing qui auto génére des figures en mouvement

Narrator & Processing
Narrator and the Temple of Processing. Cette série est composée de trois opus.

Ce deuxième volet de la saga raconte la création d'un artefact qui s'auto-génère.
Après avoir vu les bases de Processing dans le premier volet, votre narrateur voulu créer un code complexe, uniquement d'objets 3D. Ce code devait auto-générer du mouvement et des formes, avec comme seul ordre extérieur la souris (mouseX,mouseY).
